输入输出与文件操作

2020-10-07- 2020-10-16
1k- 4m-
Python > python进阶-I/O

输入

  1. python提供了input() 内置函数从标准输入读入一行文本。读入的都是字符型
  2. input() 的返回值永远是是字符串，需要强制类型转换转为自己想要的类型input()在python2.x和python3.x中的区别
int(input())
input()括号里的是提示语句。跟读取的文本没有关系。
   <!--eval()函数用来执行一个字符串表达式，并返回表达式的值--> 
​```python
sum=input("请输入")
print(type(sum))
sum=eval(sum)
print(sum)
print(type(sum))
print(isinstance(sum,int))

输入2+3，输出结果为

请输入2+3
<class 'str'>
5
<class 'int'>
True

python的内置函数中有2个类型判断函数

  • type()函数,判断是哪种类型更优。
  • isinstance()函数，判断2个参数类型是否相同更优。

输出

格式化输出

整数的输出

- %o---oct 八进制
- %d---dec 十进制
- %x---hex 十六进制
print('%o'%20)
print('%d'%20)
print('%d'%20)

输出的结果是

24
20
14

浮点数的输出

  • %f—默认小数点后保留6位，不够的用0补齐

  • %e—科学计数法，默认小数点后保留6位

  • %g—默认6位有效数字。数字长度超过所取的有效数字长度，自动转换为科学计数法

print('%f'%1.23)
1.230000
print('%.1f'%1.23)
1.2
print('%e'%1.23)
1.230000e+00
print('%.2e'%123.4)
1.23e+02
print('%g'%1.23)
1.23
print('%.2g'%123.4)
1.2e+02
  • 内置round() 函数
round(number[,ndigits])
'''param number-数字表达式
         ndigits-四舍五入后保留的位数。默认0
   return 四舍五入后小数点保留ndigits位的值'''

字符串输出

  • %s
  • %10s——右对齐，占位符10位
  • %.2s——从左边起截取2位字符

文件操作

打开文件

  • python通过内置的**open()==函数打开文件
<变量名>=open(<文件名>,<打开模式>)
文件的打开方式 描述
r 以只读方式打开文件，这是默认模式。如果不存在则产生FileNotFoundError异常
w 打开一个文件只用于写入。如果该文件已存在则打开文件，并清空之前内容。如果该文件不存在，创建新文件
x 写模式，新建一个文件，如果该文件已存在则会产生FileExistsError异常
a 打开一个文件用于追加。如果该文件已存在，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入
b 二进制模式
t 文本模式 (默认)
+ 在r/w/x/a的基础上使用打开一个文件进行更新(可读可写)
r+：在文件末尾进行追写
<f>.close() 将文件关闭

文件读取

文件的读取方式 描述
<f>.read(size=-1) 读入全部数据，给出参数，则读入前size个字节
<f>.readline(size=-1) 读入一行的数据，给出参数，则读入前size个字节
<f>.readlines(size=-1) 读入文件所有行，形成以行为元素的列表。给出参数，则读入前size个元 素
<file>.readall() 读入整个文件内容，返回一个字符串或字节流
我们在读取到了文件之后常常需要使用split()函数来分割字节流获得预期的数据。

文件写入

文件的写入方式 描述
<f>.write(str） 写入字符串str或字节流str
<f>.writeline(ls) 写入一个元素全是字符串的列表
ls=[‘你好’,’世界’]
f.writeline(ls)
你好世界
<f>.seek(offset) 改变当前文件的指针，offset有
0-文件头；1-当前位置；2-文件末尾
f.seek(0)   回到文件的开头位置

上下文管理with语句

在实际开发中应优先考虑使用上下文管理语句with，with总能保证文件正确关闭。

语法形式如下

with open(filename,mode,encoding) as fp:
'''param filename 文件名
    	 mode 打开模式
         encoding 用何种编码打开文件
    	 fp 文件对象'''

关闭文件

<file>.close()
